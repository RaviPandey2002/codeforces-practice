# codeforces-practice

100 problems, rated 800 to 1200, solved in order before entering any contest.
Each problem is picked to teach a specific idea — not random grinding.

Language: JavaScript / TypeScript

---

## Structure

```
codeforces-practice/
├── problems/       ← one folder per problem  (4A-watermelon/solution.js)
├── contests/       ← one folder per contest  (1234-div2-round-xyz/)
└── notes/          ← patterns, mistakes, anything useful
```
