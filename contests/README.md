# contests/

One folder per contest participated in (after completing the 100-problem practice track).

```
contests/
├── 2240-div2-round-1105/
│   ├── overview.md     ← what you solved, what you missed, patterns seen
│   ├── A-solution.js
│   ├── B-solution.js
│   └── ...
└── ...
```

**Naming format:** `<contest-id>-<div>-<short-name>/`

Each contest folder should include an `overview.md` written after the contest summarising:
- Problems solved during vs. upsolving
- Patterns used
- Mistakes made
- What to study next
